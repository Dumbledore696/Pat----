### B1052

字符串操作

* “Are you kidding me? @\/@”的’\’是转义字符，想要输出’\’就要用’\\’表示

* 别忘了加上a<1||b<1||c<1||d<1||e<1

* ```c++
  if(t1[i]==' '||t1[i]=='['||t1[i]==']'){
      i++;
      continue;
  }
  ```
  
  改为
  
  ```c++
  if(t1[i++]!='['){
      continue;
  }
  ```
  
  因为'[]'外的字符不只有空格
  
* 程序改进之处：改用vector<vector <string> > ,不需要用map，用string的substr截取[]中的字符串substr(pos,len)

* 注意格式正确，在Are you kidding me? @\/@ 后输出换行符。
